Fill out the form, hit Copy, paste into data/events.json on GitHub. No build step required.

Heads up. This page is publicly accessible. Do not enter sensitive info. The form output is only useful if you commit it to the repo, so nothing here writes to anything on its own.

How to use

  1. Pick "Create new event" or an existing event to edit.
  2. Fill in the fields. The JSON preview at the bottom updates as you type.
  3. Click Copy full events.json.
  4. Open data/events.json on GitHub, select all, paste, commit.
  5. GitHub Pages redeploys in roughly a minute. Share the event URL.

Could not auto-load events.json

This usually happens when you open the file directly (file://) rather than over a server. Paste the current contents of data/events.json below and click Load.
